# Fraud Detection using Machine Learning

This project builds a fraud detection system using machine learning and deploys it using a Streamlit web application.

## Dataset

The dataset used for this project contains more than **6 million financial transactions** and is used to detect fraudulent activities.

You can download the dataset from Kaggle:

Dataset Link:  
https://www.kaggle.com/datasets/amanalisiddiqui/fraud-detection-dataset?resource=download

After downloading, place the dataset file inside the **data/** folder of this project.

## Project Workflow

1. Data Exploration and Visualization
2. Handling Class Imbalance (Fraud rate ~0.13%)
3. Feature Engineering
4. Model Training using Logistic Regression
5. Model Evaluation
6. Deployment using Streamlit

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Matplotlib
- Seaborn
